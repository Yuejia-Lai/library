## 常用cmd命令

- 盘符切换

```
D:  //切换到D盘
```

- 当前路径下所有内容

```
dir
```

- 进入单级目录

```
cd filename
```

- 回退到上一级

```
cd ..
```

- 进入多级目录

```
cd 目录1\目录2\...
```

- 回退到盘符

```
cd \
```

- 清屏

```
cls
```

-  退出命令行窗口

```
exit
```

- 打开对应的文件

```
filename.xxx  //包含后缀的文件名即可
```

注：如果要快速打开某一文件，可以将文件对应的路径（不包含文件本身）加入**系统环境变量**中，这样就可以直接使用文件名在任意目录下打开该文件。
